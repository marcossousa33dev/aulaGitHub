# 🚀 Repositório de Prática - Git e GitHub

Bem-vindo! 👋  
Este repositório foi criado para **aprender na prática como utilizar Git e GitHub**, simulando um ambiente real de desenvolvimento.

---

## 🎯 Objetivo

Aqui você irá aprender:

- Criar e gerenciar repositórios
- Trabalhar com **branches**
- Fazer commits corretamente
- Enviar alterações com `push`
- Atualizar o projeto com `pull`
- Trabalhar com **Pull Requests (PR)**

---

## ⚠️ Regra principal

🚫 **NÃO é permitido dar `push` direto na branch `main`**

✔️ Todo trabalho deve ser feito em uma **branch própria**  
✔️ Depois deve ser aberto um **Pull Request**
